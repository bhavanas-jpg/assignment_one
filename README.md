## How to install and run locally?

```
$ git clone https://github.com/bhavanas-jpg/assignment_one.git
$ cd weather-forecast
$ npm install
$ npm run dev
```
<hr/>

## Features-

- Search weather of any location all over globe
- Shows humidity, temperature and conditions of weather
